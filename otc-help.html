Create A New Sku Description

Getting Started
  • Go to the Create a New Sku Page: Click Here
  • Fill out the form, click the Save Button As changes are made
  • DO NOT FORGET to do the sku part. THIS IS REQUIRED!

Exiting

Sku

This is the sku of the item that is in Stone Edge.

RULES

  • Skus cannot have:
    • Capital Letters (A, B, C, etc.)
    • Spaces
    • Special Characters other than dashes
      • YES: - (dash)
      • NO: . , ' " ! ? / \ ( ) [ ] { } @ # $ % ^ & * ; :
        (period, comma, single/double quotes, exclamation/question marks, forward/back slashes, parenthesis, brackets, curly braces, etc.)

Is a Parent, Child or Neither

  • Choose from the drop down menu if this item is a parent, child or neither.
  • If sku is a parent/child: put parent or child
  • If sku is not a parent/child: put neither

Manufacturer Information

Name
  • *Required*
  • Name of the Manufacturer
  • Example:
    • Roscoe Medical
    • Axelgaard
Phone
  • *Required*
  • Phone Number for manufacturer
  • If there are multi check:
    • Is there a toll-free and direct? Go with the toll-free (1-800...)
    • Is there one that's USA based and the others international? Go with the USA-based.
    • Is one for specifically: Customer Service? Go with that one.
    • If there is still confustion, either: see if we have other product online by them or ask.
  • Format:
    • #-###-###-#####
    • or
    • ###-###-####
Address Line 1
  • *Required*
  • Address line 1 for manufacturer
Address Line 2
  • *Optional*
  • Address line 2 for manufacturer
City
  • *Required*
  • City for Manufacturer
State
  • *Required*
  • State Abreviations ONLY
  • Example:
    • Florida: FL
    • Georgia: GA
    • Ohio: OH
  • Need Help? See Here: Reference
Zipcode
  • *Required*
  • Zip Code for Manufacturer
  • Format:
    • #####
    • 33326
    • 33323
Website URL
  • *Required*
  • Put the FULL url to the manufacturers website.
  • Example:
    • http://www.otcwholesale.com/
    • http://www.roscoemedical.com/

Paragraphs

  • *Required*
  • Main paragraph
  • Version 1 goes in Paragraph 1
  • Version 2 goes in Paragraph 2
  • If you need help, ask or see: Editor Tips

Paragraphs For

  • *Required*
  • Choose from the drop down menu if the paragraph (1 or 2) is either for:
  • OTCWholesale
  • or Payless
  • Make sure both are not the same

Features

  • a general explanation that includes things like:
    • # of channels, modes, or programs
    • power supply
    • extras that make it different (if there are any)
  • some features are in description
  • Ex: backlit screen, vanishing scent, one-touch operation, adjustable settings
  • This can be phrases or sentences
  • Avoid 'yeah phrases' i.e. - excellent product quality

Has Technical Specifications

Choose from the drop down menu if this item has technical specifications. If yes, choose Yes, or leave it as No.

Technical Specifications

  • In most items this is found in the manual, or on the manufacture website. If it's being taken from the manufacture website then it needs to be double checked just to make sure the setting are the same. Always go be what's in the manual, what is on the item or packaging, and on any brochures or flyers the manufacturer has. If there is a discrepancy between them - bring it to Fico. This may be needed to be clarified by the manufacturer.
  • This is all the technical information like modes, programs, frequency, pulse rate/width, sizing, settings, etc.
  • For other items (non-electrotherapy) its sizing, battery, power supply, etc.

Has Warnings

Choose from the drop down menu if this item has warnings. If yes, choose Yes, or leave it as No.

Warnings

  • This is an area where some items may have 5 pages of warnings in their manuals.
  • Use best judgement as some warnings are a little more important or more practical.
  • For the warnings section do not include all the warnings that are listed in the manual. Only some are needed. Keep the number under 10.
  • You can combine some to make one together. For example:
    • "Don't put electrodes on eyes. Don't put electrodes on throat. Don't put electrodes over the heart."
    • This might be listed as 3 different warning in manual but can be combined into:
    • "Don't put electrodes on eyes, throat, or over the heart."

Has Directions

Choose from the drop down menu if this item has directions. If yes, choose Yes, or leave it as No.

Directions

  • If the item has a drug label, or label with directions, put that here.
  • If the item has a manual with (a lot of) directions, put either the simplified form or leave blank.

Has Uses

Uses

  • This is sometimes used and other times deleted.
  • For items with a Drug Facts type label there is a Uses section and that is used.
  • For others there is a Uses section in a manual and may times that is summed up.
